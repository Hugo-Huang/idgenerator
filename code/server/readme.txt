一、程序名称
	分布式ID生成器
二、程序简介
	1.提供分布式ID服务，具备以下特性：
	（1）产生全局唯一、且单调递增的ID；
	（2）任何情况下ID不会重复或者回退；
	（3）具备高效率产生ID的能力；
	（4）具备提供多种ID的能力；
	（5）便于运维管理；
	2.程序架构
	采用Redis作为ID分发器，一个ID同时只能属于一个分发源（Redis实例），以保证ID的全局唯一性；工作过程中采用分段加载ID的方式，
	即每次只向Redis中加载一段ID，当这段ID使用到一定程度之后，再从数据库中加载下一段到Redis；数据库中存储每个ID的下一个可用段的信息；
